 - <b>rotational(转动)</b>：the net torque about any point be zero(外力矩为0)

when something is rotational equilibrium:the net force will all pass a common point.(当某个物体处于转动平衡时，它所受到的外力将交于某一点)(3个非平行外力)<br>
